THE PEOPLE THAT INFLUENCED THE CREATION OF THE INTERNET

J.C.Licklider

Joseph Carl Robnett Licklider came up with the idea of a global computer network. He led the defense department research, organized the funding and gathered the team which made ARPANET became a reality.The ARPANET is considered the first internet. Learn more

Paul Baran

Paul Baran wanted to create an indestructible communication system during the cold war. He envisioned developing a non-centralized switching communication system that would function even when many links and switching nodes were destroyed. He named the transportation mechanism of the nodes “hot-potato routing.” An idea he learned from the mice in a mace.

Learn more
Bob khan and Vinton Cerf

Bob Khan and Vinton Cerf were co-founders of the Transmission Control Protocol(TCP). The idea was to replace the existing network protocol, NCP, used in the ARPANET. This idea was to make computers and networks worldwide communicate with each other regardless of the hardware or software they use.

Learn more
Robert Taylor

Robert Taylor wanted to build a network that would link heterogeneous computers together. Is big idea was time-sharing and for computers to be communication devices. He hoped to build a computer network to connect the ARPA-sponsored projects together and to communicate to all of them through one terminal.

Learn more
Lawrence Robert

Bob Taylor hired Larry Robert to help realize his vision for creating a network that will link heterogeneous computers together. Larry came up with the first experimental network. He built a network between two computers, but his frustration was that he wanted to link more computers to the network using packet switching, which most computer scientists did not support.

Raymond Tomlison

As a graduate hired to help build the earliest components of Advanced Research Projects Agency Network (ARPANET), Tomlinson, at his own time, decided to build a networked messaging program using the @ symbol to indicate networked email, which is still very relevant today.

Tim Berners-Lee

He created the world wide web. According to Wikipedia, in 1989, CERN was the most prominent Internet node in Europe, and Berners-Lee saw an opportunity to join hypertext with the Internet: "I just had to take the hypertext idea and connect it to the TCP and DNS ideas and—ta-da!—the World Wide Web". said Tim Berners-Lee

Marc Andreessen

After the creation of the world wide web, the web was still line and text. It was not very user-friendly. Marc Andreessen wanted to figure out how to make the internet user-friendly and easier to use images, video, and audio capabilities. This led to the birth of web browser
